Introduction

SmartAdmin is a premium, feature‑rich Enterprise Admin Dashboard built on Symfony 7 and Bootstrap 5.3.7. It delivers a clean, modern, and fully responsive design packed with powerful components, pre‑built layouts, and advanced UI elements to help you build scalable web applications, SaaS platforms, CRMs, CMSs, or custom dashboards with ease.

With dark mode support, flexible color schemes, and a rich collection of charts, tables, and widgets, SmartAdmin lets you present data beautifully on any screen size. It’s perfect for developers and teams looking to accelerate their project workflow while maintaining enterprise‑grade quality.

Prerequisites

Please install and set up the following before proceeding:

  • PHP >= 8.x

  • Composer

    Make sure Composer is installed and running on your computer.

    Download Composer

  • Node.js

    Download and install the stable LTS version of Node.js.

    Download Node.js

  • Gulp

    Ensure that you have Gulp installed and running on your computer. If you already have Gulp installed, you can skip this step. If not, you can install it by running the following command from your terminal: npm install -g gulp@4.0.1.

  • Symfony CLI

    The Symfony CLI is a developer tool to help you build, run, and manage your Symfony applications directly from your terminal. If you already have installed Composer on your computer, you can skip this step.

    Download Symfony CLI

Installation

You can run this app using one of these package managers:

Bun, Yarn, or NPM

1. Bun (Recommended for macOS & Linux)

Install Bun with npm i -g bun or sudo npm i -g bun.

Command Description
bun i Install all dependencies.
bun dev Run the project locally.
bun run build or gulp build Create a production build.

2. Yarn

Install Yarn with npm i -g yarn or sudo npm i -g yarn.

Command Description
yarn Install all dependencies.
yarn dev Run the project locally.
yarn build or gulp build Create a production build.

3. NPM

NPM comes pre-installed with Node.js.

Command Description
npm i or npm i --force Install all dependencies.
npm run dev Run the project locally.
npm run gulp or gulp build Create a production build.

CakePHP Commands

Run the following in a separate terminal to spin up CakePHP:

Command Description
composer i Install PHP dependencies.
symfony server:start Runs the project locally, starts the development server.
